Déploiement sur plateformes mobiles

Nous avons utilisé PhoneGap en le connectant directement à notre dépôt GitHub.

Plusieurs adaptations ont été nécessaires :

  • un petit bout de javascript embarqué dans toutes les pages web pour changer dynamiquement la taille du logo sur mobile ;

  • idem pour changer la taille de la Q-Table ;

  • pour limiter les permissions demandées par l'application, on a utilisé un fichier config.xml ;

  • on a changé la taille par défaut de l'aire de jeu de 15x10 à 6x6.

Procédure détaillée :

  • Se connecter à PhoneGap avec son compte githup
